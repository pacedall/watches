# Pacedall Watches — Website v2

Premium digital watch faces for Apple Watch, Android Wear OS & Zepp/Amazfit.
A Pacedall Labs Company · Est. 2024

## Pages
- `index.html` — Homepage
- `pages/collection.html` — All watch faces with category/platform filters
- `pages/about.html` — Brand story & values
- `pages/contact.html` — Contact form

## Design System
- Colors: Navy #1B2A6B · Gold #F5A623 · Sky Blue #4A90D9 · White #FFFFFF
- Typography: Inter (UI) + Playfair Display (headings)
- Fully WCAG AA contrast compliant
- White background throughout (per Pacedall Design Guidelines v1.0)

## Collections
- Simplistic — Clean, minimal everyday faces
- Luxury — Dark dials with gold/chrome accents
- Sport — High-contrast fitness data faces
- Battery Saver — Ultra-efficient low-power designs

## Platforms
- Apple Watch (Series 4–Ultra 2, watchOS 7+)
- Android / Wear OS 3+
- Zepp OS 3 / Amazfit (GTR, GTS, T-Rex, Falcon, Band)

## Compliance
- terms.pacedall.com — linked prominently in all footers
- privacy.pacedall.com — linked prominently in all footers
- Back-to-top button on all long pages
- Sticky header on all pages
- Mobile-first responsive (44px touch targets)

## Deploy to GitHub Pages
1. Push all files to repo root
2. Settings → Pages → Source: main branch / root
3. Done — no build step required
